## Проекты

**[Link Tracker](https://github.com/EgorXx/LinkTracker)**  
Система мониторинга контента на микросервисной архитектуре.  
Bot Service + Scrapper Service, асинхронная коммуникация через Apache Kafka, паттерн Transactional Outbox, многопоточная обработка ссылок, тесты внешних API через WireMock.  
`Spring Boot` `Kafka` `PostgreSQL` `Liquibase` `WireMock`

**[TourGuRu](https://github.com/EgorXx/TrourGuRu)**  
Веб-приложение для поиска путешествий — намеренно написан без фреймворков.

`Java Servlets` `JDBC` `PostgreSQL` `Tomcat`
